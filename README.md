# Clemson Baja SAE Website

A modern, smooth website built to represent Clemson University's Baja SAE team. Designed to highlight team activites, showcase competition, enable sponsorship engagement, and encourage recruitment. 

## Overview
The Official Clemson Baja SAE Website is an entirely front-end web project built using HTML, CSS, and JavaScript, utilzing a custom design, fluid effects, and an organized structure. 

The website serves as the club's online presence, mainly featuring:
  - Team information and recruitment
  - Competition events
  - Sponsorship contributions
  - A consistent interface across all pages

## Getting started

In order to run this locally, you first need to clone the repository

```git clone https://github.com/dantemallozzi05/Clemson-Baja-Website.git```

### Run Locally

Navigate to root directory in file explorer, and open ```index.html``` in any modern browser
